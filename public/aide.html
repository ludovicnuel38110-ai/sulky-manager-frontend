Centre d’aide Sulky-Bet

Tout comprendre pour parier simplement et gérer tes gains efficacement

Comment jouer ?

  1. Va sur Dashboard → Courses
  2. Sélectionne une réunion puis une course
  3. Choisis tes chevaux
  4. Sélectionne ton type de pari
  5. Entre ta mise puis clique Parier

Types de paris

Simple gagnant

Ton cheval doit finir 1er.

Simple placé

Ton cheval doit finir dans les 3 premiers.

Couplé gagnant

Les 2 premiers chevaux.

Couplé placé

Deux chevaux parmi les 3 premiers.

Trio

Les 3 premiers chevaux.

Calcul des gains

Formule :

Gain = Mise × Cote finale

Les gains sont crédités automatiquement lorsque la course est réglée.
